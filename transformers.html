Keajaiban NLP dengan Transformers

Memanfaatkan model bahasa canggih dengan beberapa baris kode.

Contoh Penggunaan Pipeline

1. Analisis Sentimen

Contoh Output:

Kalimat: 'I love this new phone, it's amazing!'
Hasil: Label=POSITIVE, Score=0.9999

Kalimat: 'The flight was delayed and the service was terrible.'
Hasil: Label=NEGATIVE, Score=0.9998
# 1. Instal library yang dibutuhkan (jalankan di Colab/terminal)
# !pip install transformers torch

from transformers import pipeline

# 2. Muat pipeline analisis sentimen (model akan di-download otomatis)
classifier = pipeline("sentiment-analysis")

# 3. Siapkan beberapa kalimat contoh
sentences = [
    "I love this new phone, it's amazing!",
    "The flight was delayed and the service was terrible."
]

# 4. Lakukan prediksi
results = classifier(sentences)

# 5. Tampilkan hasil
for sentence, result in zip(sentences, results):
    label = result['label']
    score = result['score']
    print(f"Kalimat: '{sentence}'")
    print(f"Hasil: Label={label}, Score={score:.4f}\n")

2. Klasifikasi Zero-Shot

Contoh Output:

Urutan Label: ['teknologi', 'olahraga', 'politik']
Skor:         [0.985, 0.008, 0.007]

Kalimat ini paling relevan dengan label 'teknologi'.
# Library harus sudah terinstal dari contoh sebelumnya
from transformers import pipeline

# 1. Muat pipeline klasifikasi zero-shot
classifier = pipeline("zero-shot-classification")

# 2. Siapkan sekuens dan label kandidat
sequence_to_classify = "Apple just announced the new M4 chip for the iPad Pro."
candidate_labels = ["teknologi", "olahraga", "politik"]

# 3. Lakukan prediksi
result = classifier(sequence_to_classify, candidate_labels)

# 4. Tampilkan hasil
print(f"Kalimat: '{result['sequence']}'")
print(f"Urutan Label: {result['labels']}")
print(f"Skor:         {[round(score, 3) for score in result['scores']]}")
print("\nKalimat ini paling relevan dengan label '{}'.".format(result['labels'][0]))
